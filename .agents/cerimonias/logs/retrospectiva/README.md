# Logs da Retrospectiva

Os logs reais da **Retrospectiva** devem ser gravados nesta pasta seguindo o
padrao:

- `YYYY-MM-DD-HHMM-<branch>.md`

Cada arquivo deve nascer a partir do template em
[`../retrospectiva-template.md`](../retrospectiva-template.md) e precisa ter
link para a issue do `Jira` e para a pagina do `Confluence` quando existirem.
